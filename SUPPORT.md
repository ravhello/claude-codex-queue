# Support

## Setup and usage questions

Start a
[GitHub Discussion](https://github.com/ravhello/claude-codex-queue/discussions)
for installation help, account discovery questions, or advice about a local
configuration. Discussions make reusable answers easier to find than private
messages.

Include:

- Claude + Codex Queue version;
- Windows, WSL distribution, Python, Claude CLI, and Codex CLI versions;
- whether the source is Claude Code, Claude Desktop, Remote SSH, or Codex App;
- the exact command or UI action;
- a redacted error and the relevant final log lines.

Useful diagnostics:

```bash
claude-codex-queue doctor
claude-codex-queue status -v
```

## Bugs

Use the
[bug report form](https://github.com/ravhello/claude-codex-queue/issues/new?template=bug_report.yml)
for reproducible defects. A minimal reproduction is more useful than a full
private log.

## Security

Do not open a public issue for a vulnerability or suspected credential leak.
Use
[private vulnerability reporting](https://github.com/ravhello/claude-codex-queue/security/advisories/new)
and follow [SECURITY.md](SECURITY.md).

## Data that must be removed

Never post API keys, OAuth tokens, authentication files, email addresses,
account UUIDs, private prompts, full transcripts, or unredacted Windows paths.
